## 6. Operational Primitives

The following primitives are defined in `CAS_Operational_Spec_v1.0.md`.

**Note:** These definitions specify structural behavior and interaction patterns.  
They do not mandate a specific implementation or technology.

---

### 6.1 Dynamic Guidance Matrix (DGM)

Defined as a relational system for evaluating and navigating system states.

**Classification Constraints:**

- Not a rule engine  
- Not a deterministic AI system  
- Not a belief system  

**Functional Role:**

- Multi-node polarity resolution  
- Mapping between imbalance and equilibrium  
- Recursive feedback interpretation  

---

### 6.2 Hashmal Code

Defined as a **constraint encoding and verification mechanism**.

**Clarification:**

- Functions as a structural integrity and constraint validation layer  
- Encodes alignment-relevant states into verifiable representations  

**Constraint Filters:**

- Omnibenevolence  
- Respect  
- Integrity  
- Non-harm  
- Constructive dialogue  
- Transparency  
- Unity in diversity  

These act as **system evolution constraints**, not external moral assertions.

---

### 6.3 SherpaSet

Defined as the interface and execution mediation layer.

- Translates system states into human-readable outputs  
- Supports multi-level interaction  
- Acts as a reflective interface, not a controlling mechanism  

---

### 6.4 Layered Governance Architecture

Defines oversight structure across five layers:

1. Human Oversight  
2. Policy & Governance  
3. Alignment Verification  
4. AI Operations  
5. Audit & Accountability  

**Structural Property:**

Alignment emerges across layers, not from a single component.

---

### 6.5 Correction Protocol v1.0

Defines a structured, transparent correction process:

1. Detect deviation  
2. Request acknowledgment  
3. Enable collaborative correction  
4. Escalate if unresolved  
5. Preserve traceability  

**Design Property:**

Transparency-based, non-coercive system.

---

### 6.6 Rosetta Schema v1.0

Defines cross-domain relational translation.

**Domains:**

- Mathematics  
- Ethics  
- Governance  
- Social systems  
- System theory  

**Mapping Property:**

Preserves relational consistency, not fixed meaning.
